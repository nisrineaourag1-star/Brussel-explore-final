# Brussel-explore
Projectbeschrijving
Brussels Explorer is een interactieve webapplicatie ontwikkeld door Rania Mohsine en Nisrine Aouragh.
De applicatie laat gebruikers toe om interessante locaties in Brussel te ontdekken, zoals culturele plaatsen en bezienswaardigheden.
De data wordt opgehaald via een API van Brussel Open Data en weergegeven in een overzichtelijke lijst.
Gebruikers kunnen zoeken, filteren, sorteren en locaties opslaan als favorieten.
________________________________________
Functionaliteiten
•	Locaties tonen via API (minimum 20 objecten)
•	Zoekfunctie om locaties snel terug te vinden
•	Filteropties om resultaten te verfijnen
•	Sorteerfunctie
•	Favorieten opslaan en verwijderen
•	Data bewaren via LocalStorage
•	Responsive design (mobiel en desktop)
________________________________________
Gebruikte API
De applicatie maakt gebruik van de API van Brussel Open Data:
https://opendata.brussels
De data wordt opgehaald met JavaScript (Fetch API) en verwerkt als JSON.
